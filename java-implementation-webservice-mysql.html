关于webservice

由于之前写过JAVA和PHP的JSON以及JMI的网络服务，这次在使用webservice完成管理待办事项自然就联想到它们。JAVA和PHP JSON、JMI、webservice三者共同之处是可以提供服务器的资源共享。 Web Service具有跨平台性，可以在不同设备、不同编程语言中进行调用，所以它的应用范围应该要比RMI更广
